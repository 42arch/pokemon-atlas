### 欢迎来到宝可梦的世界
**宝可梦图谱 (Pokémon Atlas)** 是一个交互式的视觉百科全书，让你以全新的方式探索广袤的宝可梦宇宙。不再是枯燥的列表或维基百科，你将置身于一个由宝可梦、属性和特性组成的庞大星系中，直观地感受它们之间的羁绊。

### 主要功能
- **探索错综复杂的关系**：直观地查看宝可梦之间是如何通过“进化路线”、“共同属性”以及“特性（包含隐藏特性/梦特）”相互连接的。
- **快速搜索与发现**：使用内置的搜索功能，快速定位你最爱的宝可梦、属性或特性。
- **专注模式**：点击任意一个节点即可将其锁定，图谱会自动为你高亮显示所有与它有直接关联的元素，屏蔽外界干扰。
- **详尽的图鉴详情**：通过侧边栏面板，你可以轻松查看宝可梦的图鉴编号、世代信息、特性的具体效果描述，甚至是详细的进化条件（例如“升级”或“使用道具”）。
- **自定义视图**：
  - 随时打开或关闭特定类型的连线（如仅查看进化路线，或仅查看特性关系），让图谱更清晰。
  - 使用“世代”过滤器，仅显示你最熟悉的那一代宝可梦。

### 快速开始

#### 1. 安装依赖
```bash
pnpm install
```

#### 2. 准备数据
运行以下命令来构建宝可梦的数据集：
```bash
pnpm data:csv
```

#### 3. 启动应用
```bash
pnpm dev
```
在浏览器中打开 [http://localhost:3000](http://localhost:3000) 即可开始你的探索之旅！
